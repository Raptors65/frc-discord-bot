# FRC Discord Bot

A Discord bot that uses the [API from The Blue Alliance](https://www.thebluealliance.com/apidocs/v3) to access event data for the FIRST Robotics Competition.

## Commands

### Events
`/events [team number]`

### Predictions
`/rankings <event key>`

### Schedule
`/schedule [team number]`

### History
`/history [team number]`

### Bracket
`/bracket <event key>`
